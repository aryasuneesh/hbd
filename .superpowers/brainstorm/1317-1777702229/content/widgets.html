Image & Privacy Approach
Sender-uploaded photos will be stored in Firebase Storage client-side only — no server processing. Images are linked to a bouquet ID and never indexed or made publicly searchable. Senders see a disclaimer before uploading. Pinterest images are linked by URL only (no re-hosting). No images are stored by default during development — the upload UI will be present but gated behind a feature flag until storage is reviewed.
